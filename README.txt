This is an Arduino project file. It is a text file, so it can be opened in Notepad/Notepad++. It contains example code to implement your own RC Car project onto the arduinos on board our cars.

If you need to find documentation for hardware, here are the components which were used:

Truck: Traxxas Slash
Arduino: Arduino Uno Rev 3
Bluetooth Module: HC-04
Ultrasonic Sensors: HC-SR04
Gesture/Light/Color/Prox sensor: APDS9960
